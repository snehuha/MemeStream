🎭 MemeStream

A clean and interactive random meme generator built using JavaScript and a public meme API.

🚀 Features

- 🔄 Hover-based meme generation (autoplay on hover)
- 🖼️ Dynamic meme fetching using REST API
- ⏳ Loading spinner for better UX
- ⚠️ Basic error handling
- 🎨 Light/Dark mode toggle with saved preference
- ⬇️ Download meme functionality
- ✨ Smooth UI transitions and modern design
